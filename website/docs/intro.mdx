# Introdução

Nesse guia, aprenderemos como utilizar o JS SDK do Starlight, uma ferramenta oficial que automatiza o processo
de integração de sites e aplicações JavaScript com as APIs do Starlight.

{/* asdasd */}

:::tip Dica

Quer ver exemplos de uso ou lembrar rapidamente como utilizar uma API específica? Dê uma olhada na seção Cheatsheet.

:::

O JS SDK pode ser usado em qualquer aplicação JavaScript e TypeScript. Porém, os seguintes SDKs podem ser melhores

## Instalação

Antes de continuar para as próximas páginas, instale o SDK no seu projeto:

<Tabs>
  <TabItem value="npm" label="npm" default>

  ```bash
  npm install @starlightcms/js-sdk
  ```

  </TabItem>
  <TabItem value="yarn" label="yarn" default>

  ```bash
  yarn add @starlightcms/js-sdk
  ```

  </TabItem>
</Tabs>
