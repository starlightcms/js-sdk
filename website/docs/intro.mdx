# Introdução

Nesse guia, aprenderemos a utilizar o JS SDK do Starlight, uma ferramenta oficial que automatiza o processo
de integração de sites e aplicações JavaScript com as APIs do Starlight.

O JS SDK pode ser usado em qualquer aplicação JavaScript e TypeScript. Porém, se sua aplicação usa React, você pode
utilizar o Starlight React SDK, que provê componentes React que facilitam a integração com o Starlight, além de todas
as funcionalidades do JS SDK.

:::info Versão da API

O JS SDK requisita e envia dados para o Starlight utilizando a versão 2 das APIs.

:::

## Instalação

Para começar a utilizar o JS SDK, instale-o no seu projeto:

<Tabs>
  <TabItem value="npm" label="npm" default>

  ```bash
  npm install @starlightcms/js-sdk
  ```

  </TabItem>
  <TabItem value="yarn" label="yarn" default>

  ```bash
  yarn add @starlightcms/js-sdk
  ```

  </TabItem>
</Tabs>

Depois de instalado, você precisa configurar qual área de trabalho deve ser utilizada para requisitar dados. Saiba mais
na [página sobre o Cliente](./client#configurando-o-cliente-global) desse guia.

## TypeScript

O JS SDK é escrito em TypeScript e "tipa" corretamente todos os seus métodos, o que é útil mesmo que você use JavaScript
comum na sua aplicação, já que sua IDE pode usar esses tipos para lhe ajudar no desenvolvimento.

Além disso, o SDK provê uma maneira simples de criar tipos para os dados dos *modelos* e *singletons* da sua *área de trabalho*.

Para saber mais, consulte a página sobre TypeScript desse guia.
